Run your own (Python 3 + numpy)

Check any ticker not in the list (non-destructive what-if):
python3 analyze_name.py --name "Nebius" --growth 0.65 --rev_bn 1.5 --tam_bn 200 --tam_cagr 0.40 --fwd_pe 90 --beta 2.1 --quality 0.6 --earn_quality 0.4 --dominance 0.4 --capex_elastic 0.4

Other tools: stress_test.py (macro ladder), correlation.py (concentration), build_dashboard.py (rebuild the views). Full instructions in the guide.
One package: open this page, then pick what you need. Calibration files (calibration.py, tam_bridge.py, tiering.py, engine.py) are the model owner's — please don't edit; send corrections instead.
